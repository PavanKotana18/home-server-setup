Home Server Services

This document lists the open-source services planned for the home server.

---

Pi-hole

Purpose: Network-wide ad blocking

Repository
https://github.com/pi-hole/pi-hole

Pi-hole blocks advertisements and tracking domains for all devices in the home network using DNS filtering.

---

Unbound

Purpose: Local DNS resolver

Repository
https://github.com/NLnetLabs/unbound

Unbound performs recursive DNS resolution locally, improving privacy and reducing reliance on external DNS providers.

---

Nextcloud

Purpose: Personal cloud storage

Repository
https://github.com/nextcloud/server

Nextcloud provides file storage, synchronization, and sharing similar to services like Google Drive.

---

Jellyfin

Purpose: Media streaming server

Repository
https://github.com/jellyfin/jellyfin

Jellyfin allows streaming movies, shows, and other media stored on the home server.

---

Vaultwarden

Purpose: Self-hosted password manager

Repository
https://github.com/dani-garcia/vaultwarden

Vaultwarden is a lightweight self-hosted implementation compatible with Bitwarden clients.

---

Uptime Kuma

Purpose: Service monitoring

Repository
https://github.com/louislam/uptime-kuma

Uptime Kuma monitors server services and provides status dashboards.

---

WireGuard

Purpose: Secure remote access VPN

Repository
https://github.com/WireGuard/wireguard

WireGuard is a modern, fast, and secure VPN protocol used to access the home server remotely without exposing services directly to the internet.
