Security Practices

This document lists the security practices planned for the home server.

VPN Access

All services will be accessed through a VPN connection using WireGuard.

No services will be exposed directly to the public internet.

Strong Passwords

All accounts and services will use strong passwords with at least 12–16 characters.

Two-Factor Authentication

Two-factor authentication will be enabled for services that support it.

System Updates

The server will be updated regularly using:

sudo apt update
sudo apt upgrade

Router Security

Router security measures include:

- Changing the default admin password
- Disabling WPS
- Keeping router firmware updated
